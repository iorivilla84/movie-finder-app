# movie-finder-app

A lightweight web application that displays detailed movie information in a modal window. Users can view genres, runtime, ratings, and available streaming providers (Rent, Buy, Watch it now). Movie data is fetched dynamically via HTTP requests from an external API, allowing for up-to-date and scalable content rendering. Built using HTML5, modern CSS with nesting (Sass-like), and vanilla ES6 JavaScript. Auto-deployed via Netlify.
